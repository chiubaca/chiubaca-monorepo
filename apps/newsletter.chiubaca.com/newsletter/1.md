## 💻 Tech News

**[Wake up babe, another React server component vulnerability dropped](https://x.com/CloudflareDev/status/2052467567588196703)** and [Kent C. Dodds](https://x.com/kentcdodds/status/2052541237656539271) has officially checked out of RSC as a concept. Kent's always been a measured voice in the React ecosystem. His take: _"I'm relieved that I never actually adopted RSC and at this point I'm convinced I never will."_.

If you're on a framework thats using RSC make sure you're patched up. Maybe consider jumping to Tanstack Sta...

**[Tanstack ecosystem got hacked](https://tanstack.com/blog/npm-supply-chain-compromise-postmortem)** - Sh\*t , IS NO WHERE SAFE!? Attackers poisoned TanStack’s GitHub Actions cache via a malicious PR, then abused trusted npm publishing tokens to ship malware-infected packages that stole developer secrets and spread through the supply chain. Once again, everyones looking at npm and thinking of [xkcd #2347](https://xkcd.com/2347/).

Anyways... Hows about some more React news? **[Fate 1.0 — The First Full Async React Metaframework](https://x.com/cnakazawa/status/2054920522488852790)**. Christoph Nakazawa shipped Fate 1.0 and is calling it the first full Async React metaframework; zero-config live views, Drizzle support, and a "Void Router" that sounds like it belongs in a mecha anime. Do I think this looks cool though? Hell yeah. Do I have time to build another todo list app with this? Hell no!

Moving on.

Nobody... Absolutely no one... Not a single soul of earth... **[Bun: Lets rewrite in Rust!](https://github.com/oven-sh/bun/pull/30412)**

Jarred Sumner, the absolute nutter, shows us what a cracked dev with unlimited tokens can do and out of nowhere drop a PR that vibe refactors Bun from Zig to Rust. From the PR description, **It shrinks the binary by 3–8 MB, fixes memory leaks, and most importantly: compiler-assisted tools to catch memory bugs that have cost them years. No async Rust, same architecture, same data structures**. Sounds good, so why is the upvote/downvote ratio on this one _so_ _divisive_!? Is vibe coding critical infrastructure still not acceptable yet?

And talking of burning tokens, OpenClaw maintainer Peter Stoiber, just [casually spent 1.3 mil](https://x.com/steipete/status/2055346265869721905) on tokens on,**check notes**, automation!?

## 👀 Things That Caught My Eye

**[Datatype](https://franktisellano.github.io/datatype/)** — A font that renders characters as charts. Type `12345`, get a bar chart. Type a letter, get a sparkline. It's a real "wait, fonts can do _what?_" moment. Could be a fun way to visualise issue stats without reaching for D3.

**[Pokémon TCG Archive](https://www.cosmos.so/wilk/pokemon-tcg-archive/canvas)** — Nearly 20,000 Pokémon TCG cards archived as high-res images on Cosmos and honestly it's the most dangerous thing you could show a sleep-deprived dad with a credit card and a nostalgia problem.

**[Claude Code /radio Command](https://x.com/DataChaz/status/2053018292806160387)** — Claude Code now has a `/radio` fires up Lo-Fi beats.
