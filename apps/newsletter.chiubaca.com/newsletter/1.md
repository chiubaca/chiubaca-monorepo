## 👀 Things That Caught My Eye

**[Datatype](https://franktisellano.github.io/datatype/)** — A font that renders characters as charts. Type `12345`, get a bar chart. Type a letter, get a sparkline. It's one of those "wait, fonts can do *what?*" moments. I've already bookmarked it for a "by the numbers" footer on this very site. Could be a fun way to visualise issue stats without reaching for D3. (Not that I have time to learn D3. I have a baby.)

## 🎮 Currently Playing

**Pragmata** — Capcom's strangest release in years, and I'm completely hooked. It's an action game with a hacking mechanic woven into combat in a way that actually *feels* great, not a QTE, but a system you learn and exploit. Capcom are on an incredible run right now. This comes right off the back of completing **Resident Evil Requiem**, which was also excellent. If you have any love for experimental action games, Pragmata is worth your time.
