## 👀 Things That Caught My Eye

**[Datatype](https://franktisellano.github.io/datatype/)** — A font that renders characters as charts. Type `12345`, get a bar chart. Type a letter, get a sparkline. It's one of those "wait, fonts can do *what?*" moments. I've already bookmarked it for a "by the numbers" footer on this very site. Could be a fun way to visualise issue stats without reaching for D3. (Not that I have time to learn D3. I have a baby.)

**[React Server Components: Another Vulnerability Drop](https://blog.cloudflare.com/react-server-components-vulnerabilities/)** — CVEs disclosed affecting RSC and Next.js. Cloudflare patched their WAF but the deeper issue remains: every new serialization boundary is a new surface area. I don't use Next.js (Astro supremacy etc.) but watching the discourse is fascinating. The RSC skeptics are getting louder and the defenders are getting quieter.

**[Kent C. Dodds Taps Out on RSC](https://x.com/kentcdodds/status/2052541237656539271)** — This is significant. Kent's been a measured voice in the React ecosystem. His take: *"I'm relieved that I never actually adopted RSC and at this point I'm convinced I never will."* When someone of his stature publicly bails, it's a signal. As one commenter put it: *"the memes were right - we really are redoing the first 20 years of PHP again with zero collective memory."*

**[Star Fox — 4v4 Online Dogfights](https://x.com/NintendoAmerica/status/2052184135951626441)** — Star Fox is back with online multiplayer on Switch 2 and honestly? About time. 4v4 dogfights, GameShare over GameChat — Nintendo's finally giving Arwing combat the treatment it deserves. I haven't been this excited about a Star Fox since... well, since I had free time. (Calvin sleeps in 45-minute windows. I'll take what I can get.)

**[Cloudflare on RSC Vulnerabilities](https://x.com/CloudflareDev/status/2052467567588196703)** — Wake up baby, another server component vulnerability dropped.

**[Fate 1.0 — The First Full Async React Metaframework](https://x.com/cnakazawa/status/2054920522488852790)** — Christoph Nakazawa shipped Fate 1.0 and is calling it the first full Async React metaframework — zero-config live views, Drizzle support, and a "Void Router" that sounds like it belongs in a mecha anime.

**[Pokémon TCG Archive](https://www.cosmos.so/wilk/pokemon-tcg-archive/canvas)** — Nearly 20,000 Pokémon TCG cards archived as high-res images on Cosmos and honestly it's the most dangerous thing you could show a sleep-deprived dad with a credit card and a nostalgia problem.

**[Bun Rewritten in Rust](https://github.com/oven-sh/bun/pull/30412)** — Jarred Sumner dropped a PR rewriting Bun in Rust, shrinks the binary by 3–8 MB, fixes memory leaks, and most importantly: compiler-assisted tools to catch memory bugs that have cost them years. No async Rust, same architecture, same data structures. The upvote/downvote ratio on this one is *divisive*.

**[CodexBar](https://x.com/steipete/status/2055346265869721905)** — OpenClaw maintainer spent 1.3 mil on tokens.

## 🎮 Currently Playing

**Pragmata** — Capcom's strangest release in years, and I'm completely hooked. It's an action game with a hacking mechanic woven into combat in a way that actually *feels* like hacking — not a minigame, not a QTE, but a system you learn and exploit. Capcom are on an incredible run right now. This comes right off the back of completing **Resident Evil Requiem**, which was also excellent. If you have any love for experimental action games, Pragmata is worth your time. *(Played mostly in 45-minute sessions while Calvin naps. The save system is merciful.)*
