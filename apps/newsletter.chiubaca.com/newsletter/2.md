**[Liquid DOM](https://x.com/AndrewPrifer/status/2056923983581446529)** — Andrew Prifer rebuilt Apple's Liquid Glass design language for the web, but wait, [Jhey](https://x.com/jh3yy/status/2063214567937692013?s=20) already did this a year ago!? So who wore it better?

**[Hyper](https://hyperjs.ai/)** — Image ShadCN but for the your backend. that exactly what Hyper is. An API framework for Bun that ditches npm entirely, it copies TypeScript source directly into your repo under `src/hyper/`. One route definition spits out runtime, OpenAPI, typed RPC client, and an MCP server. With the number of supply chain attacks on the rise, maybe this is not a terrible idea...

**[DeepSeek Makes 75% Price Cut on V4 Pro Permanent](https://www.bloomberg.com/news/articles/2026-04-27/deepseek-slashes-fees-for-new-ai-model-in-chinese-price-war)** — DeepSeek makes its 75% price cut on V4 Pro permanent — In: $0.435/M, Out: $0.87/M. That's ~1/35 the price of GPT-5.5. Either Huawei Ascend chips are that cheap or DeepSeek's inference optimization is genuinely next-level. Then coming in the with one-two, they also dropped **[Reasonix](https://esengine.github.io/DeepSeek-Reasonix/)** a native coding agent built on DeepSeek. Chinse Claude Code is here and its looking dangerous for Anthropic.
