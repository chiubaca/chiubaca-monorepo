## 👀 Things That Caught My Eye



**[Modern Web Guidance](https://developer.chrome.com/docs/modern-web-guidance)** — Chrome released an "AI coding agent skills" package — basically prompts and patterns to make your coding assistant build modern, accessible, performant web apps instead of the usual slop. Works with Claude Code, Copilot, Vercel's agent. Install it as a plugin and your AI stops writing 2015-era jQuery.

**[Liquid DOM](https://x.com/AndrewPrifer/status/2056923983581446529)** — Andrew Prifer rebuilt Apple's Liquid Glass design language for the web — shape morphing, refraction, dispersion, full HTML integration, and a Three.js bridge. It looks like actual magic.

**[Hyper](https://hyperjs.ai/)** — API framework for Bun that ditches npm entirely — it copies TypeScript source directly into your repo under `src/hyper/`. One route definition spits out runtime, OpenAPI, typed RPC client, and an MCP server. It's like if Rails and vendoring had a baby, but fast.

**[DeepSeek Makes 75% Price Cut on V4 Pro Permanent](https://www.bloomberg.com/news/articles/2026-04-27/deepseek-slashes-fees-for-new-ai-model-in-chinese-price-war)** — DeepSeek makes its 75% price cut on V4 Pro permanent — In: $0.435/M, Out: $0.87/M. That's ~1/35 the price of GPT-5.5. Either Huawei Ascend chips are that cheap or DeepSeek's inference optimization is genuinely next-level. Probably both.

**[Reasonix](https://esengine.github.io/DeepSeek-Reasonix/)** — DeepSeek also shipped Reasonix, a native coding agent built on DeepSeek models with aggressive caching and low cost. Terminal-first, looks like Cursor but born in Beijing.

**[Fresh](https://getfresh.dev/)** — Really want to like Fresh TUI, but it's just not sticking. I think I'm too deep into VS Code.

**[npm `allowScripts`](https://github.com/npm/cli/pull/9360)** — npm is finally making install scripts opt-in by default. Phase 1 adds `allowScripts` to your root `package.json`, `npm approve-scripts <pkg>` for opting in, and identity matching via lockfile URLs so attackers can't swap in a forged package and skip detection.

**[Resident Evil Veronica - Announcement Trailer | PS5 Games](https://youtu.be/S4msqGQxSAg)** — Wait, they're actually doing it? Resident Evil Code: Veronica is getting the remake treatment and I genuinely didn't think Capcom had it in them. GIMME.

---

**[Astro 6.4 — Pluggable Markdown Pipeline](https://astro.build/blog/astro-640)** — Three big ones in this drop: a swappable `markdown.processor` API (goodbye top-level `remarkPlugins`, hello `unified({...})`), an experimental Rust-based Markdown processor called Sätteri that shaved over a minute off Astro's own docs build, and Cloudflare `cf()` helpers for the advanced routing experiment. The Sätteri thing is wild — no `remark`/`rehype` support yet, but if they make it the default in a future major? That's a proper escape hatch from the unified ecosystem. Tempted to flip it on for this newsletter and measure the difference. (Though let's be honest, two Markdown files isn't exactly a stress test.)

**[@chiubaca on X](https://x.com/chiubaca/status/2062641385795182598)** — I called it! Cloudflare acquiring VoidZero was una-VOID-able. (Yes I'm quoting my own tweet from March. Somebody has to.)
